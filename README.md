# Mason Lavinder - Portfolio Website

Personal portfolio website showcasing my work as a Developer.

**Live Site:** [https://masonlavinder.github.io/lavinder/](https://masonlavinder.github.io/lavinder/)


## Tech Stack

- React + TypeScript
- Vite
- CSS3

## Development

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Deployment

Deploy to GitHub Pages:
```bash
npm run predeploy
npm run deploy
```
